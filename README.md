### Цель работы:
Разработка программного инструмента на языке Python для автоматизированного создания тестовых сценариев для программ на языке C++ посредством статического анализа исходного кода

### Задачи:
1. Изучить основные подходы к тестированию программного обеспечения
2. Проанализировать существующие инструменты автоматической генерации тестов
3. Изучить методы анализа исходного кода
4. Разработать модуль парсинга C++ кода с целью извлечения информации о функциях.
5. Реализовать алгоритм генерации тестовых случаев на основе полученных данных о функциях.
6. Разработать механизм формирования тестов в формате фреймворка Google Test.
7. Провести тестирование разработанного решения.
8. Проанализировать полученные результаты и оценить эффективность разработанного решения.
