### Обзор предметной области

Сущетсвует довольно много инструментов для анализа сложности кода на C++. Приведу несколько наиболее популярных примеров:
1. Klocwork - очень мощный инстремент статического тестирования безопасности приложений (SAST) и анализа качества кода в реальном времени. Он используется в сферах, где цена ошибок очень велика. Это платный инструмент с высоким порогом входа;
2. SonarQube - популярная платформа с открытым исходным кодом для непрерывного анализа и обеспечения качества программного кода. Она автоматически проверяет код на наличие багов и уязвимостей безопасности. Анализ сложности коде более поверхностный;
3. Lizard - простой, но мощный инструмент статического анализа кода, который измеряет цикломатическую сложность, количество строк и количество параметров функций. Он написан на Python, поэтому так популярен.

### Ожидаемый результат
В рамках данной работы планируется разработка инструмента, осуществляющего анализ сложности кода на C++.

Разрабатываемый инструмент должен быть простым в установке и использовании, что позволит применять его как в учебных, так и в практических задачах.

##### Основной функционал инструмента включает:
1. Выполнение анализа исходного кода на основе абстрактного синтаксического дерева (AST);
2. Выделение ключевых конструкций (функции, условия, ветвления);
3. Расчёт метрик сложности для каждой функции, а также для файла исходного кода в целом;
4. Формирование отчёта в формате JSON (с возможной последующей визуализацией в формате HTML).

Такой отчёт позволит пользователю выявлять участки кода повышенной сложности и принимать решения о необходимости его упрощения или рефакторинга.
